# TI-QUEST
A simple RPG game for the TI-84 Plus CE

How to upload to your calculator:
  1) Download TI Connect™ CE Software
    https://education.ti.com/download/en/ed-tech/CA9C74CAD02440A69FDC7189D7E1B6C2/EE9E51FA137C4F388252DD191CAB37BF/TIConnectCE-6.0.3.619.exe
  2) Open software
  3) Connect the calculator via a mini usb-b cable
  4) Drag and drop the TIQUEST.8xp file to connected calculator
Running the program:
  1) Go to your calculator and press the "PRGM" button
  2) Scroll down to "TIQUEST" using the arrow buttons
  3) Press "ENTER"
  4) Press "ENTER" again to run the program
Playing the game:
  1) Watch https://youtu.be/yzcg7YWqKao?si=P3K1hOXZ8YB4T4Xv to see the original tutorial by Freckled Fox.
  2) Note that this game has some differences.
    A. Move your character [x] using the buttons 8, 4, 2, and 6 on the number pad
    B. Press "clear" to quit
    C. Pixie dust lets you time-travel instead of teleporting
    D. There is neither a telescope nor love potion in this game
    E. You get infinite inventory
    F. You can shop for all items at the same time
  3) KEY:
    [+] Attack
    [F] Food
    [{#}] Woo
    [ * ] Health
    [A] Treasure
    [π] Shop
    [θ] Monster
    [ * ] Ally
    [0] Obstacle
